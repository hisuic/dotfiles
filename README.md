# Requirements

## Basic system requirements

| Item | Name |
| -------------- | --------------- |
| OS | Linux |
| Desktop Environment | Hyprland |
| Terminal| Kitty |
| Shell| bash |


## Required Packages
- kitty
- waybar
- dunst
- neofetch/fastfetch
- mako
- rofi


# Other Requirements

## Nvim
Get my Nvim config at [here](https://github.com/hisuic/nvim).

## Wallpaper
Get my wallpaper collection at [here](https://github.com/hisuic/walls).

## Prompt Theme Engine
I use [Oh My Posh](https://ohmyposh.dev/). My config is available at 'assets/poshthemes/hisuic.omp.json'.

## hspool(Optional)
Awesome pocket command dictionary on wayland.
Recommended to you when you use hyprland with my config!
Get hspool [here](https://github.com/hisuic/hspool).
